# Dark Mode Color Mapping

This document maps light mode colors to dark mode equivalents for systematic conversion.

## Background Colors
- `bg-white` → `bg-gray-800`
- `bg-gray-50` → `bg-gray-900`
- `bg-gray-100` → `bg-gray-800`
- `bg-gray-200` → `bg-gray-700`
- `bg-gray-300` → `bg-gray-600`

## Text Colors
- `text-gray-900` → `text-gray-100`
- `text-gray-800` → `text-gray-200`
- `text-gray-700` → `text-gray-300`
- `text-gray-600` → `text-gray-400`
- `text-gray-500` → `text-gray-400`

## Border Colors
- `border-gray-200` → `border-gray-700`
- `border-gray-300` → `border-gray-600`

## Colored Backgrounds (Status/Info)
- `bg-blue-50` → `bg-blue-900/20`
- `bg-blue-100` → `bg-blue-800/30`
- `bg-green-50` → `bg-green-900/20`
- `bg-green-100` → `bg-green-800/30`
- `bg-yellow-50` → `bg-yellow-900/20`
- `bg-yellow-100` → `bg-yellow-800/30`
- `bg-red-50` → `bg-red-900/20`
- `bg-red-100` → `bg-red-800/30`
- `bg-purple-50` → `bg-purple-900/20`
- `bg-purple-100` → `bg-purple-800/30`

## Colored Text
- `text-blue-700` → `text-blue-300`
- `text-blue-800` → `text-blue-200`
- `text-green-700` → `text-green-300`
- `text-green-800` → `text-green-200`
- Keep vibrant colors as-is: `text-blue-600`, `text-green-600`, etc.

## Hover States
- `hover:bg-gray-50` → `hover:bg-gray-700`
- `hover:bg-gray-100` → `hover:bg-gray-700`
